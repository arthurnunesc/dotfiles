# Yeet

Commit the current changes, separating commits into human-readable logical chunks.

Follow the commit title convention used by the repository. Prefer Conventional Commits when no stronger local convention exists. Push to the relevant branch. If the correct branch is unclear, ask which branch before pushing.

Do not include AI attribution or `Co-Authored-By` trailers.
